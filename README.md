## Installation

1. Download the MSI for your Revit version from [Releases](https://github.com/AmorphousDev/amorphous-releases/releases/latest)
2. Run the installer (no admin rights required)
3. Open Revit — find the **Amorphous** tab in the ribbon

## Add-ins

### Section Box Tools

| Add-in | Description |
|--------|-------------|
| **Smart Section Box** | Create section boxes around selected elements with automatic rotation alignment. Supports linked models, in-place families, curtain walls, and model groups. Includes padding adjustment panel with undo/redo history. |
| **Edge-Aligned Section Box** | Create section boxes aligned to a picked edge — ideal for angled geometry. |
| **Section Box Manager** | Save, load, and share section box configurations. Import/export for team collaboration. Stores up to 99 configs per project. |
| **2D View from Section Box** | Generate section/elevation views from the current section box boundaries. |
| **Align 3D to Section Box** | Orient the 3D view camera to align with section box faces. |

### View Tools

| Add-in | Description |
|--------|-------------|
| **QuickVis** | One-click visibility toggles for categories, links, and point clouds. Save/restore visibility snapshots. Minimal mode for presentations. |
| **Interactive View Range** | Live sliders to adjust view range (cut plane, top, bottom) with real-time preview. |
| **Rotate Active View** | Rotate plan views to any angle. Useful for angled buildings or site orientations. |
| **Align View to Edge** | Rotate the active view to align with a picked edge. |
| **2D View from Picked Face** | Create a section or elevation view perpendicular to any picked face. |
| **Toggle Point Clouds** | Quick on/off toggle for point cloud visibility in the active view. |
| **Zoom to Selected** | Zoom and center the view on selected elements. Works across viewports. |

### UI Selection

| Add-in | Description |
|--------|-------------|
| **Smart Selection** | Select elements by type, family, category, workset, or phase. Supports region selection and linked models. |
| **Isolate by Filter** | Temporarily isolate elements by type, family, category, workset, or phase. Toggle behavior — run again with no selection to exit isolation. |
| **Hide by Filter** | Temporarily hide elements using the same filter options. Toggle to unhide. |

### Graphics

| Add-in | Description |
|--------|-------------|
| **Color by Filter** | Apply color overrides by category, type, family, workset, or phase. Includes QA Checker mode for finding sloped pipes, duplicate elements, and more. |
| **VG Manager** | Bulk visibility/graphics management across multiple views. |

### Family Tools

| Add-in | Description |
|--------|-------------|
| **Export Families** | Batch export families from the project to external .rfa files. Filter by category. |
| **In-Place to Loadable** | Convert in-place families to loadable families. Preserves geometry and materials. |

### Data Tools

| Add-in | Description |
|--------|-------------|
| **View Template Manager** | Audit view templates with usage counts, compare templates side-by-side, purge unused, and export to Excel. |

### Settings

| Add-in | Description |
|--------|-------------|
| **Settings** | Account management, privacy settings, check for updates, and add-in preferences. |
